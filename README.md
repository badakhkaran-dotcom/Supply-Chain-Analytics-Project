# Supply-Chain-Analytics-Project
Project Overview
This project focuses on analyzing supply chain and procurement operations to improve supplier performance monitoring, production planning, inventory visibility, and procurement decision-making.

The solution integrates MySQL, Python (Pandas & Numpy), and Power BI to transform raw supply chain data into actionable business insights through KPI dashboards, SQL analytics, and data visualization.

****Business Problem**
Organizations often struggle with:
Limited visibility into supplier performance
High supplier dependency risks
Procurement inefficiencies
Delivery delays affecting production
Increasing defect rates and quality issues
Lack of centralized KPI reporting

This project addresses these challenges by developing a Supply Chain Procurement & Supplier Performance Analytics System.

**Key Business Objectives**
Monitor supplier-wise procurement performance
Analyze product-wise revenue and order trends
Track lead times and delivery delays
Identify high-risk suppliers
Improve inventory and production planning
Optimize procurement and logistics operations
Support data-driven business decisions
